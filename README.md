## 🇬🇧 English

**PlutoMail** is a modern, fast, and secure disposable email service client built with React and Vite. It utilizes the [Mail.tm](https://mail.tm/) API to provide users with instant, temporary email addresses to protect their privacy and avoid spam.

### ✨ Features
- **🚀 Instant Generation:** Get a temporary email address in seconds.
- **🔄 Auto-Refresh & Polling:** Inbox automatically updates.
- **📎 Attachment Support:** Download attachments and preview inline images seamlessly.
- **🌙 Dark Mode:** Built-in dark/light mode toggle that respects your system preferences.
- **📋 Copy to Clipboard:** One-click email copy functionality.
- **🎨 Modern UI:** A beautiful, responsive, and space-themed interface built with Tailwind CSS.

### 🛠️ Technologies Used
- **Frontend Framework:** React 18, Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **API:** mail.tm API

### 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/plutomail.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd plutomail
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   ```

## 🇪🇸 Español

**PlutoMail** es un cliente para servicio de correo electrónico temporal y desechable, moderno, rápido y seguro, construido con React y Vite. Utiliza la API de [Mail.tm](https://mail.tm/) para proporcionar a los usuarios direcciones de correo principal temporales al instante y así proteger su privacidad y evitar el spam.

### ✨ Características
- **🚀 Generación Instantánea:** Obtén una dirección de correo temporal en segundos.
- **🔄 Actualización Automática:** La bandeja de entrada se actualiza sola automáticamente.
- **📎 Soporte de Adjuntos:** Descarga archivos adjuntos y previsualiza imágenes integradas dentro del correo.
- **🌙 Modo Oscuro:** Opción de modo oscuro/claro que respeta las preferencias de tu sistema.
- **📋 Copiar al Portapapeles:** Funcionalidad para copiar la dirección con un solo clic.
- **🎨 Interfaz Moderna:** Una interfaz hermosa, responsiva y con temática espacial construida con Tailwind CSS.

### 🛠️ Tecnologías Utilizadas
- **Framework Frontend:** React 18, Vite
- **Estilos:** Tailwind CSS
- **Iconos:** Lucide React
- **API:** API de mail.tm

### 🚀 Cómo Empezar

1. **Clona el repositorio:**
   ```bash

   ```
2. **Navega al directorio:**
   ```bash
   cd plutomail
   ```
