## 🇪🇸 Español

**PlutoMail** es un cliente de correo electrónico temporal moderno y minimalista, desarrollado con React y Vite. Consumiendo la API de [Mail.tm](https://mail.tm/), permite generar direcciones desechables al instante para proteger la privacidad del usuario, evitar el spam y mantener limpias las bandejas de entrada personales.

**Privacy-First & Clean UX:** A diferencia de otras alternativas en el mercado, PlutoMail está diseñado para respetar el enfoque del usuario. Es una herramienta completamente gratuita, sin cuotas de uso, sin barreras ocultas y estrictamente libre de anuncios y rastreadores.

### 🌐 Aplicación en Vivo
La aplicación ya está desplegada y lista para usar. Puedes acceder directamente en:  
**[https://plutomail.vercel.app/](https://plutomail.vercel.app/)**

### ✨ Características Principales
- **🚀 Generación Efímera:** Creación de casillas seguras en cuestión de segundos y con un solo clic.
- **🔄 Sincronización Automática:** Sistema de *polling* integrado que actualiza la bandeja de entrada en tiempo real sin necesidad de recargar la página.
- **📎 Gestión de Adjuntos:** Soporte nativo para la descarga de archivos y renderizado seguro de imágenes *inline* dentro del cuerpo del correo.
- **🎨 UI/UX Moderna y Temática:** Interfaz responsiva con temática espacial construida en Tailwind CSS.
- **🌙 Tema Dinámico:** Soporte completo para Dark/Light mode que hereda automáticamente las preferencias de tu sistema operativo.

### 🛠️ Stack Tecnológico
- **Frontend:** React 18, Vite
- **Estilos e UI:** Tailwind CSS, Lucide React (Iconografía)
- **Backend/Servicios:** API de mail.tm

## 🇬🇧 English

**PlutoMail** is a modern, minimalist disposable email client built with React and Vite. By leveraging the [Mail.tm](https://mail.tm/) API, it generates instant temporary addresses to protect user privacy, keep spam at bay, and maintain clean personal inboxes.

**Privacy-First & Clean UX:** Unlike many alternatives on the market, PlutoMail is designed to respect the user's focus. It is an entirely free tool with no usage quotas, no hidden paywalls, and is strictly ad-free and tracker-free.

### 🌐 Live Application
The application is deployed and ready to use. You can access it directly at:  
**[https://plutomail.vercel.app/](https://plutomail.vercel.app/)**

### ✨ Key Features
- **🚀 Ephemeral Generation:** Secure inbox creation in seconds with a single click.
- **🔄 Real-Time Sync:** Built-in polling system that automatically updates the inbox without requiring manual page refreshes.
- **📎 Attachment Handling:** Native support for downloading files and securely rendering inline images within the email body.
- **🎨 Modern, Themed UI/UX:** Responsive, space-themed interface built seamlessly with Tailwind CSS.
- **🌙 Dynamic Theming:** Full support for Dark/Light mode that automatically inherits your operating system's preferences.

### 🛠️ Tech Stack
- **Frontend:** React 18, Vite
- **Styling & UI:** Tailwind CSS, Lucide React (Icons)
- **Backend/Services:** mail.tm API
